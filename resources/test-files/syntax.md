# Markdown Syntax Guide

## Headings
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

## Emphasis
*Italic* or _Italic_
**Bold** or __Bold__
***Bold and Italic***

## Lists
### Unordered List
- Item 1
- Item 2
    - Subitem 2.1
    - Subitem 2.2

### Ordered List
1. Item 1
2. Item 2
    1. Subitem 2.1
    2. Subitem 2.2

## Links
[Link Text](https://example.com)

## Images
![Alt Text](https://example.com/image.png)

## Code
### Inline Code
`Inline code`

### Code Block
```markdown
Code block
```

## Blockquotes
> This is a blockquote.
>> Nested blockquote.

## Horizontal Rule
---

## Task Lists
- [ ] Task 1
- [x] Task 2
